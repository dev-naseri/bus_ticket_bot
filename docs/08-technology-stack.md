# 08 - Technology Stack

## Overview
- Explain why technology stack is important for the project.
- Mention that the stack is chosen for simplicity, maintainability, and alignment with project goals.

---

## Programming Language
- Python
- Reason: Widely used, easy to integrate with Telegram API, and good support for database operations.

---

## Database
- SQLite
- Reason: Lightweight, easy to set up, suitable for local and small-to-medium scale applications.
- ORM: Peewee (for easy object-relational mapping)

---

## Telegram API
- pyTelegramBotAPI (Telebot)
- Reason: Provides simple and effective access to Telegram Bot API.

---

## Libraries / Modules

- peewee
- pyTelegramBotAPI
- khayyam
- Any additional libraries required for date handling, validation, etc.
